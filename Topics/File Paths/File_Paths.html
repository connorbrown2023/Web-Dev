File Paths

This topics goes over different file paths and will use images within the folder show about different file paths

There are two types of file paths absolute and relative

  • Absolute:

    • This is the full file path to a spesific location, all the way from the C: drive

  • Relative:

    • This is file path is relative to the folder that the image is in, so looking at the paths notice how it shows specifces what folder it is in before using the name of the image, it doesnt need the full file path

In this topic i have used both the absolute and relative file paths
